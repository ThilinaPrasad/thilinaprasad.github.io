## Usage

<Step number={1} title="Create an account on Sendgrid" />

- Create an account on [sendgrid](https://signup.sendgrid.com/).
- Generate an [API KEY](https://app.sendgrid.com/guide/integrate/langs/nodejs)

<Step number={2} title="Authenticate sender" />

Sendgrid requires you to authenticate yourself (email account) which you want the emails to be sent from. For that they have clearly mentioned instructions.

- Go to [Sender Authentication](https://app.sendgrid.com/settings/sender_auth) page.
- Create a new sender.
- Fill out the form with all your details.
- Verify by checking your mail and clicking on Verify button.

<Step number={3} title="Set environment variable" />

In your `.env` file, create an entry for `SENDGRID_API_KEY` and paste the credentials you retrieved from the sendgrid portal.

**NOTE**: DO NOT push this file to GitHub.

<Step number={4} title="Install @sendgrid/mail node package" />

- `npm install @sendgrid/mail`
- We use two methods from the package: `sendgrid.setApiKey()` and `sendgrid.send()`

A live demo of this API can be found on my [homepage](https://manuarora.in). I've used Sendgrid on my `Contact` component on the Home Page and Blog page.
